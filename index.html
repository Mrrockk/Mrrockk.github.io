# 本站介绍

本站收集了2021年往后大量的大厂笔试真题。

创建该网站的目的在于积累、收集大厂的笔试真题，如果你也有这样的疑惑——笔试题看都看不懂；刷了不少题目，但是一到笔试还是只能做一两题；做笔试等于坐牢2小时.....，那么恭喜你，发现了宝藏！本网站将带你了解近些年大厂的笔试的真实难度。无论你是大一大二的萌新、还是正在参加招牌的大三、研二的同学，相信本网站的题目都会让你有所收获，不容错过~

TIP

如果你完全没有刷过算法题，这里的题目对于你来说可能难度过大了，请先提升基础~

前言

  • 由于题目均为同学笔试后回忆而来，所以代码可能并没有进行大量的测试，如果题解有歧义，欢迎联系我们讨论~【希望友好讨论】

  • 代码以 C++、Java和Python为主，其他语言的同学也不用担心，算法的核心在于思想而不是代码实现。

  • 做算法题要注重模板，也就是一些固定的代码需要大家积累下来，不需要每次都重新构思，比如当我们需要用一个Dijkstra的时候，并不是每次都靠着自己的记忆去回忆算法过程再转换成代码，比较推荐的方式是直接记下来

TIP

如果你希望看到更多更新的笔试真题，请一定要关注我们的公众号，会实时推送最新的笔试真题~ 同时也希望推荐给你身边的同学~
